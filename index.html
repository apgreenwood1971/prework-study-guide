HTML

  • The DOCTYPE element tells the browser the document type to expect when it's opened
  • The html element is always defined in the beginning element and will contain attributes such as language. For example "en" is English
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • git status: checks what branch we are currently on
  • git checkout -b (branch name): creates a new branch and switches to it. Use feature/(name of feature) as best practice
  • git add -A: adds all changes in the current working branch to the staging area
  • git commit -m "code to be committed": stipulates what will be added/updated in the branch being worked on
  • git pull: fetch and merge

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
